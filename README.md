# EDA ROLLER COASTER ANALYSIS

Exploratory Data Analysis on roller coaster dataset using Python, Pandas, Matplotlib,  NumPy and Seaborn.

## PROJECT OVERVIEW
This project performs Exploratory Data Analysis (EDA) on roller coaster data.

## Dataset Information
- Source: Kaggle Roller Coaster Dataset
- Columns used:
  - coaster_name
  - Location
  - Status
  - Manufacturer
  - Speed_mph
- Data cleaning performed:
  - Removed missing values
  - Renamed columns
  - Filtered invalid speeds
 
## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
